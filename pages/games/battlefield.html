Battlefield 2042

Plongez dans une guerre totale futuriste !

A propos de Battlefield 2042

Battlefield 2042 est un First Person Shooter développé par les studio DICE. C'est un jeu qui ne se joue qu'en multijoueur. En effet, aucun mode campagne n'a été prévu au lancement du jeu. Ainsi les joueurs peuvent jouer sur 3 principaux modes de jeu: All Out Warfare où l'on peut retrouver les mode de jeux classiques à Battlefield (la conquête et la percée), Hazard Zone où les joueurs sont largués sur une carte du multijoueur dans le but de récupérer un maximum de cartes puis s'extraire; Les morts sont éliminatoires, c'est une sorte de Battle Royale. Et enfin le dernier mode portal, complétement nouveau qui est un mode où les joueurs peuvent créer des parties personnalisées avec leurs propres paramètres et la possibilité de jouer sur des cartes d'anciens Battlefield, avec d'anciennes armes et même d'anciens personnages.

Informations supplémentaires

Configuration recommandée:

  • Système d'exploitation : Windows 10 64 bits
  • Processeur (AMD) : AMD Ryzen 7 2700X
  • Processeur (Intel) : Intel Core i7 4790
  • Mémoire RAM : 16 Go
  • Mémoire vidéo : 8 Go
  • Carte graphique (NVIDIA) : Nvidia GeForce RTX 3060
  • Carte graphique (AMD) : AMD Radeon RX 6600 XT
  • DirectX : 12
  • Connexion en ligne : connexion Internet 512 Ko/s ou plus rapide
  • Espace libre disque dur : 100 Go SSD
